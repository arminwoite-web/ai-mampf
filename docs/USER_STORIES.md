# User Stories

## Story 1: Rezeptvorschläge basierend auf KI-Analyse erhalten
**Als** Kochinteressierter
**Möchte ich** KI-generierte Rezeptvorschläge sehen
**Damit** ich neue und kreative Gerichte entdecken kann, die auf meinen Präferenzen basieren.

**Akzeptanzkriterien:**
- [ ] Der Benutzer kann die App öffnen und eine Liste von KI-generierten Rezepten sehen.
- [ ] Jedes Rezept zeigt einen Titel, eine kurze Beschreibung und ein Bild (falls verfügbar).
- [ ] Die Rezeptvorschläge sind bei jedem Aufruf dynamisch und variieren.

**Priorität:** Hoch

## Story 2: Vegetarische Rezepte filtern
**Als** Vegetarier
**Möchte ich** Rezepte nach "vegetarisch" filtern können
**Damit** ich nur Gerichte angezeigt bekomme, die meinen Ernährungspräferenzen entsprechen.

**Akzeptanzkriterien:**
- [ ] Es gibt eine Filteroption "Vegetarisch" in der Rezeptübersicht.
- [ ] Wenn der Filter "Vegetarisch" aktiviert ist, werden nur vegetarische Rezepte angezeigt.
- [ ] Der Filter kann jederzeit wieder deaktiviert werden.

**Priorität:** Hoch

## Story 3: Glutenfreie Rezepte filtern
**Als** Person mit Glutenunverträglichkeit
**Möchte ich** Rezepte nach "glutenfrei" filtern können
**Damit** ich nur Gerichte angezeigt bekomme, die ich ohne Bedenken zubereiten und essen kann.

**Akzeptanzkriterien:**
- [ ] Es gibt eine Filteroption "Glutenfrei" in der Rezeptübersicht.
- [ ] Wenn der Filter "Glutenfrei" aktiviert ist, werden nur glutenfreie Rezepte angezeigt.
- [ ] Der Filter kann jederzeit wieder deaktiviert werden.

**Priorität:** Hoch

## Story 4: Rezepte detailliert ansehen
**Als** Kochinteressierter
**Möchte ich** die Details eines ausgewählten Rezepts einsehen
**Damit** ich alle notwendigen Informationen für die Zubereitung erhalte.

**Akzeptanzkriterien:**
- [ ] Beim Tippen auf ein Rezept in der Übersicht wird eine Detailansicht geöffnet.
- [ ] Die Detailansicht zeigt den Rezepttitel, eine ausführliche Beschreibung, Zutatenliste und Zubereitungsschritte.
- [ ] Die Detailansicht ist übersichtlich strukturiert und gut lesbar.

**Priorität:** Hoch

## Story 5: Rezept bewerten
**Als** Benutzer
**Möchte ich** ein Rezept mit 1 bis 5 Sternen bewerten können
**Damit** ich meine Meinung zu einem Rezept teilen und anderen Nutzern helfen kann, gute Rezepte zu finden.

**Akzeptanzkriterien:**
- [ ] Auf der Rezeptdetailseite gibt es eine Möglichkeit, Sterne von 1 bis 5 auszuwählen.
- [ ] Nach der Auswahl der Sterne wird die Bewertung gespeichert und dem Rezept zugeordnet.
- [ ] Der Benutzer kann seine eigene Bewertung nachträglich ändern.
- [ ] Die durchschnittliche Bewertung eines Rezepts wird angezeigt (optional, für zukünftige Iterationen).

**Priorität:** Mittel
