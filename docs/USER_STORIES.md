# User Stories

## Story 1: Rezeptvorschläge auf Basis von KI erhalten
**Als** Kochbegeisterter
**Möchte ich** KI-generierte Rezeptvorschläge sehen
**Damit** ich neue und inspirierende Gerichte entdecken kann.

**Akzeptanzkriterien:**
- [ ] Die App zeigt beim Start eine Liste von KI-generierten Rezeptvorschlägen an.
- [ ] Jeder Rezeptvorschlag enthält einen Titel, eine kurze Beschreibung und ein Bild (falls verfügbar).
- [ ] Beim Antippen eines Rezeptvorschlags wird die detaillierte Rezeptansicht geöffnet.

**Priorität:** Hoch

## Story 2: Detaillierte Rezeptinformationen einsehen
**Als** Nutzer, der ein Rezept ausprobieren möchte
**Möchte ich** detaillierte Informationen zu einem ausgewählten Rezept sehen
**Damit** ich das Gericht erfolgreich zubereiten kann.

**Akzeptanzkriterien:**
- [ ] Die detaillierte Rezeptansicht zeigt den Titel, eine Zutatenliste mit Mengenangaben und eine Schritt-für-Schritt-Anleitung.
- [ ] Die detaillierte Rezeptansicht zeigt die geschätzte Zubereitungszeit und Portionsgröße an.
- [ ] Es gibt eine Option, das Rezept zu speichern oder zu teilen.

**Priorität:** Hoch

## Story 3: Rezepte nach "vegetarisch" filtern
**Als** Nutzer mit Ernährungspräferenzen
**Möchte ich** Rezepte nach "vegetarisch" filtern können
**Damit** ich nur vegetarische Gerichte angezeigt bekomme.

**Akzeptanzkriterien:**
- [ ] Es gibt eine Filteroption "Vegetarisch" in der Rezeptübersicht.
- [ ] Wenn der Filter "Vegetarisch" aktiviert ist, werden nur vegetarische Rezepte angezeigt.
- [ ] Der Filter kann jederzeit deaktiviert werden, um wieder alle Rezepte zu sehen.

**Priorität:** Hoch

## Story 4: Rezepte nach "glutenfrei" filtern
**Als** Nutzer mit Ernährungspräferenzen
**Möchte ich** Rezepte nach "glutenfrei" filtern können
**Damit** ich nur glutenfreie Gerichte angezeigt bekomme.

**Akzeptanzkriterien:**
- [ ] Es gibt eine Filteroption "Glutenfrei" in der Rezeptübersicht.
- [ ] Wenn der Filter "Glutenfrei" aktiviert ist, werden nur glutenfreie Rezepte angezeigt.
- [ ] Der Filter kann jederzeit deaktiviert werden, um wieder alle Rezepte zu sehen.

**Priorität:** Hoch

## Story 5: Mehrere Filter gleichzeitig anwenden
**Als** Nutzer mit speziellen Ernährungsbedürfnissen
**Möchte ich** die Filter "vegetarisch" und "glutenfrei" gleichzeitig anwenden können
**Damit** ich Rezepte finde, die sowohl vegetarisch als auch glutenfrei sind.

**Akzeptanzkriterien:**
- [ ] Es ist möglich, den Filter "Vegetarisch" und "Glutenfrei" gleichzeitig zu aktivieren.
- [ ] Wenn beide Filter aktiv sind, werden nur Rezepte angezeigt, die beide Kriterien erfüllen.
- [ ] Die Filter können unabhängig voneinander aktiviert und deaktiviert werden.

**Priorität:** Mittel
